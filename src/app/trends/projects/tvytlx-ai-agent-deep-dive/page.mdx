# AI Agent Deep Dive

A research repository containing in-depth source code analysis reports of AI agents, plus a minimal educational Python agent implementation for learning core architectural patterns.

## What it does

- **Research reports**: PDF analyses of popular AI agent frameworks (ClaudeCode, Hermes Agent)
- **Educational agent**: Minimal Python implementation demonstrating core agent structure
- **Teaching materials**: Documentation focused on agent main loops, skill discovery, and CLI interfaces
- **Modular design**: Clean separation between LLM interface, skills system, and agent core

## Installation

This project uses Poetry for dependency management:

```bash
git clone https://github.com/tvytlx/ai-agent-deep-dive.git
cd ai-agent-deep-dive
poetry install
```

## Basic usage

Run the minimal teaching agent:

```bash
# Basic interaction
poetry run agt "你好"

# List available skills
poetry run agt --skills-dir ./skills --list-skills
```

<Callout type="info">
The current implementation uses a fake LLM for educational purposes. It returns test responses in streaming format to demonstrate the interface structure before connecting real model APIs.
</Callout>
