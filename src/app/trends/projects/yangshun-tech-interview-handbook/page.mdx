# Tech Interview Handbook

Tech Interview Handbook is a curated collection of coding interview preparation materials designed for busy software engineers. It consolidates essential interview content into one comprehensive resource, covering everything from algorithms to behavioral questions.

## Key Features

- **Grind 75** - Enhanced version of the famous Blind 75 LeetCode problems
- **Algorithm cheatsheets** organized by topic with practical tips
- **Behavioral interview questions** from top tech companies
- **Resume preparation guide** tailored for software engineering roles
- **Coding interview best practices** with clear do's and don'ts
- **System design interview preparation** resources
- **Front-end specific interview content** (separate handbook)
- **Study plans** with recommended practice questions

## Installation

The handbook is available as a website - no installation required:

```bash
# Visit the website
https://www.techinterviewhandbook.org/

# Or clone the repository for local access
git clone https://github.com/yangshun/tech-interview-handbook.git
```

## Basic Usage

Navigate to specific sections based on your preparation needs:

```typescript
// Example study approach
1. Start with the Software Engineering Interview Guide
2. Follow the Coding Interview Study Plan
3. Practice Grind 75 problems systematically
4. Review algorithm cheatsheets by topic
5. Prepare behavioral responses using provided frameworks
```

<Callout type="tip">
The author recommends focusing on question patterns rather than memorizing individual solutions. This approach is more efficient than grinding hundreds of random LeetCode problems.
</Callout>
