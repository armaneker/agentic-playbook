# FreeDomain

FreeDomain by DigitalPlat is a free domain registration service that provides domain names at no cost. Started as a DNS experiment and now serves over 400,000 registered domains across multiple extensions.

## Key Features

- **Free domain registration** across multiple TLD extensions (.DPDNS.ORG, .US.KG, .QZZ.IO, .XX.KG)
- **DNS provider flexibility** - works with Cloudflare, FreeDNS, Hostry, and others
- **No hidden costs** - completely free with no strings attached
- **Community support** via Telegram and Discord
- **Abuse reporting system** for maintaining platform integrity

## How to Get Started

Registration is handled through the web dashboard:

1. Visit the [DigitalPlat FreeDomain Dashboard](https://dash.domain.digitalplat.org/)
2. Choose from available extensions
3. Register your desired domain name
4. Configure with your preferred DNS provider

<Callout type="info">
Check the [tutorial documentation](https://github.com/DigitalPlatDev/FreeDomain/blob/main/documents/tutorial/index.md) for detailed setup instructions.
</Callout>

## Basic Usage

After registration, point your domain to any DNS provider:

```bash
# Example: Configure with Cloudflare
# 1. Add domain to Cloudflare dashboard

# 3. Configure DNS records as needed
```

The service acts as a domain registrar - you maintain full control over DNS configuration through your chosen provider.
