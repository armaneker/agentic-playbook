# OpenYak Desktop

OpenYak is a local-first AI desktop assistant that runs entirely on your machine. It provides office automation, data analysis, and workflow management capabilities without sending your data to the cloud.

## Key Features

- **100+ AI models** — Access latest models through OpenRouter including Claude Opus 4.6, DeepSeek V3.2, GPT-4.1
- **20+ API providers** — BYOK support for OpenAI, Anthropic, Google, Groq, and Chinese providers
- **Office automation** — Batch file operations, document generation, data analysis
- **IM integration** — Connect WhatsApp, Discord, Telegram, Slack, Signal, iMessage via OpenClaw
- **7 specialized agent modes** — Building, planning, exploring with multi-step tool calling
- **MCP connector support** — Integrate external tools via Model Context Protocol
- **Long-term memory** — Remembers preferences and context across conversations locally
- **Cross-platform** — Windows and macOS with remote access via secure tunnel

## Installation

Download the installer from the [official website](https://open-yak.com/download/) for Windows or macOS. The application runs as a standalone desktop app with no additional dependencies required.

<Callout type="info">
OpenYak offers 1M tokens/week on free models to get started. You can upgrade to premium models or bring your own API keys from 20+ providers.
</Callout>

## Basic Usage

After installation, connect a model provider and start with simple file operations:

```bash
# Ask Yak to organize files
"Clean up my Downloads folder and sort files by type"

# Analyze local data
"Parse this CSV and show me sales trends for Q4"

# Generate documents
"Turn these meeting notes into a professional summary"
```

The assistant handles file operations, data analysis, and document generation locally without uploading data externally.
