# My Brain Is Full Crew

A team of 8+ AI agents that manage your Obsidian vault through natural conversation. Built for researchers and anyone dealing with information overload, brain fog, or overwhelmed working memory.

## Key Features

- **Conversational interface** — Talk to Claude instead of managing files manually
- **8 specialized agents** — Architect, Scribe, Sorter, Seeker, Connector, Librarian, Transcriber, and Postman
- **13 workflow skills** — Complex multi-step operations like onboarding, email triage, and vault maintenance
- **Custom agent creation** — Build personalized agents for your specific problems without coding
- **Multi-language support** — Works in any language you speak naturally
- **Email and calendar integration** — Connects Gmail/Hey.com and Google Calendar to your vault
- **Audio transcription** — Converts recordings into structured meeting notes
- **Automated organization** — Agents coordinate to file, connect, and maintain your knowledge base

## Installation

<Callout type="warning">
Requires Obsidian, Claude access, and willingness to accept the project's Terms of Use during onboarding.
</Callout>

```bash
git clone https://github.com/gnekt/My-Brain-Is-Full-Crew.git
cd My-Brain-Is-Full-Crew
./setup.sh
```

The Architect agent will guide you through vault setup and configuration.

## Basic Usage

Start by talking to Claude in your Obsidian vault:

```
"I just had a meeting about the Q2 budget. Here's the recording..."
# Transcriber creates structured notes automatically

"Find everything I have about machine learning from last month"
# Seeker searches and synthesizes with citations

"My inbox is a disaster. Can you help?"
# Postman triages emails and creates action items
```

The dispatcher routes your requests to the appropriate agent or skill automatically.
