# AI Engineering Interview Questions

A curated collection of interview questions and answers covering modern AI engineering concepts. This repository serves as a reference for engineers preparing for roles in AI, LLM engineering, and agentic systems.

## Key Features

- **Comprehensive coverage** - Questions span LLM fundamentals, RAG, fine-tuning, vector databases, and AI agents
- **Role-specific preparation** - Covers AI Engineer, Gen AI Engineer, LLMOps Engineer, and AI Solutions Architect positions
- **Technical depth** - Includes both theoretical concepts and practical implementation scenarios
- **Current topics** - Covers emerging areas like agentic AI, MCP (Model Control Protocol), and quantization
- **Scenario-based questions** - Real-world problem-solving scenarios you'll encounter in production

## Installation

No installation required. Clone the repository to access the content:

```bash
git clone https://github.com/amitshekhariitbhu/ai-engineering-interview-questions.git
cd ai-engineering-interview-questions
```

## Basic Usage

The questions are organized by topic in the main README. Navigate to specific sections based on your interview focus:

```markdown
# Core areas to review
- LLM Fundamentals (transformers, attention, tokenization)
- RAG (retrieval patterns, vector search)
- Fine-tuning and Model Adaptation
- AI Agents and Agentic Systems
- LLMOps and Production AI
```

<Callout type="tip">
Focus on the "Must Know" section first - it covers LLM, RAG, MCP, Agent, Fine-tuning, and Quantization fundamentals.
</Callout>

<Callout type="info">
Many answers link to detailed explanations and video content from Outcome School for deeper understanding.
</Callout>
