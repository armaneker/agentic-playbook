# Awesome Self-hosted

A curated list of free software network services and web applications you can host on your own servers. This repository serves as a comprehensive directory for developers and system administrators seeking alternatives to SaaS providers.

## Key Features

- **Extensive categorization** — Over 70 categories from analytics to wikis
- **Quality curation** — Only free/open-source software with active maintenance checks
- **Multiple formats** — Available as HTML website and markdown repository
- **Community-driven** — Automated link checking and project maintenance validation
- **Detailed information** — Each entry includes descriptions, licenses, and implementation languages

## Installation

No installation required. Browse the curated list directly:

```bash
# Clone the repository for offline browsing
git clone https://github.com/awesome-selfhosted/awesome-selfhosted.git
```

<Callout type="tip">
The HTML version at awesome-selfhosted.net provides better navigation and search capabilities than the markdown version.
</Callout>

## Usage

Browse categories to find software that matches your needs:

```markdown
# Example: Finding a media server
Navigate to: Media Streaming > Video Streaming
Options include: Jellyfin, Plex alternatives, Emby, etc.

# Each entry provides:
- Project description
- License type  
- Implementation language
- Demo links (when available)
```
