# colleague-skill

colleague-skill transforms departing colleagues into AI skills by analyzing their communications, documents, and work patterns. It preserves their technical knowledge, work style, and personality quirks in a callable AI skill format.

## Key Features

- **Multi-source data collection** - Extracts from Feishu, DingTalk, Slack, WeChat, emails, PDFs, and screenshots
- **Dual-layer architecture** - Separates work capabilities from personality traits for realistic interactions
- **Automated skill generation** - Creates callable AI skills with colleague's coding style, decision patterns, and communication quirks
- **Evolution mechanisms** - Skills improve through additional data and conversational corrections
- **Version management** - Rollback support for skill iterations

## Basic Usage

In Claude Code, run:

```bash
/create-colleague
```

Follow prompts to input colleague name, role (e.g., "ByteDance 2-1 Backend Engineer"), personality tags, and select data sources. All fields are optional.

Once generated, invoke the skill:

```bash
/{slug}              # Full skill (persona + work)
/{slug}-work         # Work capabilities only
/{slug}-persona      # Personality only
/list-colleagues     # List all generated skills
```

<Callout type="tip">
Higher quality input data (chat logs + long documents) produces more accurate skills than manual descriptions alone.
</Callout>
