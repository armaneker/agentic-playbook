# ATLAS - Self-Improving AI Trading Agents

ATLAS is a framework for autonomous AI trading agents that improve their own prompts through market feedback. The system combines Karpathy's autoresearch methodology with Soros's reflexivity theory to create agents that evolve their trading strategies based on performance outcomes.

## Key Features

- **Multi-layer agent architecture** - 25+ agents across macro, sector, superinvestor, and decision layers
- **Self-improving prompts** - Agents modify their own prompts based on Sharpe ratio feedback
- **Autonomous agent spawning** - Creates new specialist agents when knowledge gaps are detected
- **Regime-specific training** - Separate cohorts trained on different market conditions
- **Reflexivity modeling** - Simulates how market actions change market fundamentals
- **Swarm simulation integration** - Uses MiroFish for parallel scenario generation

## Basic Usage

```python
from atlas import TradingSystem

# Initialize the system with 25 base agents
system = TradingSystem()

# Run autoresearch loop
system.evolve_agents(days=5)

# Check agent performance
system.get_agent_rankings()

# Generate trading recommendations  
recommendations = system.get_decisions()
```

The system automatically identifies the worst-performing agent by rolling Sharpe ratio, generates a targeted prompt modification, runs for 5 trading days, then either commits or reverts the change based on performance improvement.
