# AERIS-10 Open Source Phased Array Radar

AERIS-10 is an open-source, low-cost 10.5 GHz phased array radar system featuring Pulse Linear Frequency Modulated (PLFM) modulation. It's designed for researchers, drone developers, and advanced SDR enthusiasts who want to experiment with phased array radar technology.

## Key Features

- **Dual range options**: AERIS-10N (3km) and AERIS-10E (20km) versions
- **Full electronic beam steering**: ±45° in elevation and azimuth
- **Advanced signal processing**: On-board FPGA handles pulse compression, Doppler FFT, MTI, and CFAR
- **Complete hardware design**: Schematics, PCB layouts, and 3D printable enclosures
- **Python GUI**: Real-time target plotting with map integration
- **GPS/IMU integration**: Position and attitude correction
- **Modular architecture**: Separate power, frequency synthesis, and RF boards

## Basic Usage

```python
# Initialize radar system
from aeris10 import RadarSystem

radar = RadarSystem(version="N")  # or "E" for extended range
radar.initialize()

# Configure beam steering
radar.set_beam_angle(azimuth=0, elevation=10)

# Start scanning
radar.start_scan()
targets = radar.get_targets()

# Process detections
for target in targets:
    print(f"Target at {target.range}km, {target.bearing}°")
```

<Callout type="warning">
This radar operates at 10.5 GHz with significant RF power output. Ensure proper safety measures and regulatory compliance before operation.
</Callout>
