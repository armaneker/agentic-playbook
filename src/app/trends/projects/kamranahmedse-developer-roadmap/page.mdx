# Developer Roadmap

Developer Roadmap is an open-source educational platform that provides interactive roadmaps and guides for software development career paths. It offers structured learning paths across multiple technologies and roles, helping developers navigate their career progression.

## Key Features

- **Interactive roadmaps** for 60+ technologies and career paths (frontend, backend, DevOps, AI, etc.)
- **Best practices guides** covering performance, security, and code review
- **Knowledge assessment questions** to test understanding
- **Community-driven content** with regular updates
- **Visual learning paths** with clickable nodes for detailed explanations
- **Beginner-friendly variants** for complex roadmaps

## Installation

Clone and run the development server:

```bash
git clone https://github.com/kamranahmedse/developer-roadmap.git
cd developer-roadmap
npm install
npm run dev
```

<Callout type="info">
The project is primarily a web platform - most users access it directly at roadmap.sh rather than running locally.
</Callout>

## Basic Usage

Access roadmaps through the web interface:

```bash
# View all available roadmaps
open https://roadmap.sh/roadmaps

# Access a specific roadmap (e.g., React)
open https://roadmap.sh/react

# Test knowledge with questions
open https://roadmap.sh/questions/javascript
```

Click on roadmap nodes to expand topics and access detailed explanations, resources, and learning materials.
