# System Design Primer

A curated collection of resources and study materials for learning large-scale system design. This repository serves as both an educational guide and interview preparation tool for system design concepts.

## Key Features

- **Comprehensive study guide** covering scalability, performance, consistency, and availability patterns
- **Interview preparation materials** with common system design questions and sample solutions
- **Anki flashcard decks** for spaced repetition learning of key concepts
- **Real-world examples** including designs for Twitter, Uber, Netflix, and other major systems
- **Multi-language support** with translations in 15+ languages
- **Object-oriented design section** covering design patterns and principles

## Installation

Clone the repository to access all materials:

```bash
git clone https://github.com/donnemartin/system-design-primer.git
cd system-design-primer
```

For flashcards, install [Anki](https://apps.ankiweb.net/) and import the provided decks:
- `resources/flash_cards/System Design.apkg`
- `resources/flash_cards/System Design Exercises.apkg`  
- `resources/flash_cards/OO Design.apkg`

## Basic Usage

Start with the study guide in the main README, which covers fundamental concepts like:

```
1. Performance vs Scalability
2. Latency vs Throughput  
3. Availability vs Consistency (CAP Theorem)
4. Database patterns (SQL/NoSQL, sharding, replication)
5. Caching strategies
6. Load balancing
```

<Callout type="tip">
The repository includes step-by-step approaches for tackling system design interview questions, starting with requirement gathering and ending with detailed architecture diagrams.
</Callout>
