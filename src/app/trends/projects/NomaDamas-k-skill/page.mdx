# k-skill

k-skill is a comprehensive skill collection designed for Korean users to automate daily tasks through AI agents. It provides tools for booking SRT/KTX trains, sending KakaoTalk messages, checking air quality, and dozens of other Korea-specific services.

## Key Features

- **Transportation**: SRT/KTX train booking with anti-bot protection, Seoul subway arrival times
- **Communication**: KakaoTalk Mac CLI for message automation
- **Daily Life**: Fine dust monitoring, postal code search, lottery number checking
- **Entertainment**: KBO/K-League game results, nearby Blue Ribbon restaurants and bars
- **Finance**: Toss Securities portfolio management, used car price lookups
- **Shopping**: Coupang product search, Daiso inventory checking, delivery tracking
- **Documents**: HWP file processing and conversion
- **Legal**: Korean law and regulation search

<Callout type="info">
Compatible with Claude Code, Codex, OpenCode, OpenClaw/ClawHub, and other coding agents. No additional client API layer required.
</Callout>

## Basic Usage

Most skills can be invoked directly by AI agents. For services requiring authentication:

```javascript
// Example: Check KBO game results (no auth required)
const results = await kboResults.getGamesByDate('2024-04-15');

// Example: Book SRT train (requires credentials)
const booking = await srtBooking.searchTrains({
  departure: '수서',
  arrival: '부산',
  date: '2024-04-20'
});
```

Some features use proxy servers for API access without additional setup.
