# Understand-Anything

Understand-Anything is a Claude Code plugin that transforms codebases into interactive knowledge graphs. It analyzes your project using multi-agent pipelines and creates explorable visual maps of files, functions, classes, and dependencies.

## Key Features

- **Interactive knowledge graphs** with React Flow visualization showing code relationships
- **Plain-English explanations** generated by LLM for every function and class
- **Multi-platform support** across Claude Code, Codex, Cursor, and other AI coding platforms
- **Guided tours** that walk through architecture in dependency order
- **Semantic search** to find code by meaning, not just name matching
- **Diff impact analysis** showing ripple effects of changes before commit
- **Layer visualization** with automatic grouping by architectural patterns
- **Multi-agent pipeline** with specialized scanners for different analysis tasks

## Installation

### Claude Code (Native)
```bash
/plugin marketplace add Lum1104/Understand-Anything
/plugin install understand-anything
```

### Other Platforms
For Codex, Cursor, OpenCode, and other platforms, the repository includes platform-specific installation instructions in dedicated directories (`.codex/`, `.cursor-plugin/`, etc.).

<Callout type="info">
Cursor auto-discovers the plugin when you clone the repository — no manual installation required.
</Callout>

## Basic Usage

Analyze your codebase and generate the knowledge graph:
```bash
/understand
```

Open the interactive dashboard:
```bash
/understand-dashboard
```

Ask questions about the codebase:
```bash
/understand-chat How does authentication work?
```

Analyze impact of current changes:
```bash
/understand-diff
```

<Callout type="tip">
The multi-agent pipeline runs file analyzers in parallel (up to 3 concurrent) and supports incremental updates for large codebases.
</Callout>
