# Codebase to Course

A Claude Code skill that converts any codebase into an interactive, single-page HTML course designed for non-technical developers who build with AI tools.

## Key Features

- **Code-to-English translations** — Real code alongside plain-language explanations
- **Animated visualizations** — Data flow animations and component interactions
- **Interactive quizzes** — Test practical application, not memorization
- **Scroll-based navigation** — Progress tracking with keyboard shortcuts
- **Glossary tooltips** — Hover definitions for technical terms
- **Self-contained output** — Single HTML file, no dependencies

<Callout type="info">
Designed for "vibe coders" who want to understand how their AI-built projects actually work under the hood.
</Callout>

## Installation

Copy the skill into your Claude Code environment:

```bash
cp -r codebase-to-course ~/.claude/skills/
```

## Basic Usage

Open any project in Claude Code and use trigger phrases:

```
"Turn this codebase into an interactive course"
"Explain this codebase interactively"
"Make a course from this project"
```

The skill generates a complete HTML course file with embedded CSS and JavaScript.

<Callout type="tip">
Courses focus on tracing actual user flows through the code rather than abstract concepts.
</Callout>
