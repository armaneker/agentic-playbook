# Awesome AutoResearch

A curated collection of AutoResearch use cases with optimization traces and open source implementations. This repository showcases real-world applications of autonomous code optimization across different domains.

## What it does

- **Curated AutoResearch examples** - Real implementations with optimization traces showing what agents actually tried
- **Cross-domain applications** - GPU kernels, LLM training, template engines, sports analytics, and more
- **Implementation variants** - Ports for different platforms (Apple Silicon, Windows RTX, distributed systems)
- **Verification focus** - Every entry includes progress charts or full exploration traces
- **Community contributions** - Open submission process for new use cases

## Installation

This is a documentation repository, not installable software. Clone to browse locally:

```bash
git clone https://github.com/WecoAI/awesome-autoresearch.git
cd awesome-autoresearch
```

## Basic usage

Browse the repository to find relevant AutoResearch implementations:

1. Check the **Use Cases** table for domain-specific applications
2. Review **optimization traces** to understand agent behavior  
3. Explore **Implementations & Forks** for platform-specific versions
4. Follow links to source repositories for actual code

```markdown
# Example entry format
| Use Case | Description | Author | Links | Traces |
|----------|-------------|--------|-------|--------|
| Your optimization | Brief description + results | You | GitHub · Tweet | progress chart |
```

<Callout type="info">
Each entry requires verifiable optimization traces - either progress charts or full exploration logs showing the agent's decision process.
</Callout>
