# Awesome Autoresearch

A curated index of autonomous improvement loops, research agents, and systems inspired by Karpathy's autoresearch. This collection catalogs tools that implement self-improving agents capable of iterative optimization across various domains.

## Key Features

• **General-purpose descendants** - Tools that generalize autoresearch patterns to any measurable goal
• **Research agent systems** - Specialized implementations for scientific discovery and experimentation  
• **Domain-specific adaptations** - Focused implementations for coding, ML engineering, and other specific use cases
• **Platform ports** - Cross-platform implementations with hardware detection and coordination features
• **Evaluation frameworks** - Benchmarks and metrics for autonomous research systems
• **Community resources** - Notable use cases, writeups, and related research

## Installation

This is a curated list repository - no installation required. Browse the collection directly on GitHub:

```bash
git clone https://github.com/alvinunreal/awesome-autoresearch.git
cd awesome-autoresearch
```

<Callout type="tip">
Each listed project has its own installation instructions. Check individual repositories for setup details.
</Callout>

## Basic Usage

Navigate the categorized list to find implementations matching your needs:

```markdown
# Browse by category
- 🛠️ General-purpose descendants
- 🔬 Research-agent systems  
- 💻 Platform ports and hardware forks
- 🎯 Domain-specific adaptations
```

Each entry includes GitHub stars, descriptions, and links to source repositories. Many projects offer Claude Code skills, CLI tools, or platform integrations.

<Callout type="info">
Popular implementations include generalized autoresearch loops, Gemini CLI skills with Google Search grounding, and collaborative multi-agent coordination systems.
</Callout>
