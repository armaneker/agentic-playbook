# What is Agentic Playbook?

Agentic Playbook is a collection of tested, hands-on guides for setting up and running AI agent systems in production. No marketing fluff, no theory — just real commands, real configs, and real troubleshooting.

## Who is this for?

- **Developers** building multi-agent systems
- **Teams** deploying AI agents for internal workflows
- **Builders** evaluating tools like OpenClaw, Paperclip, or LangSmith Fleet

## What makes these guides different?

Every guide here is:

1. **Tested end-to-end** — every command was run before publishing
2. **Self-contained** — follow from zero to working result
3. **Honest about failures** — includes troubleshooting for real problems
4. **Timestamped** — `lastTested` date so you know if it's current
