# Anthropic Cybersecurity Skills

Anthropic Cybersecurity Skills provides 753+ structured cybersecurity skills for AI agents following the agentskills.io open standard. Each skill is mapped to MITRE ATT&CK frameworks and NIST CSF 2.0 for standardized security operations.

- **753+ cybersecurity skills** across 38 domains including penetration testing, DFIR, malware analysis, cloud security, and threat intelligence

- **Universal compatibility** with Claude Code, GitHub Copilot, Cursor, OpenAI Codex CLI, and 20+ AI platforms
- **Structured YAML frontmatter** for fast skill discovery and execution

## Basic Usage

Skills follow the agentskills.io format with YAML frontmatter and structured Markdown:

```yaml
---
name: "AWS S3 Bucket Security Audit"
category: "cloud-security"
mitre_attack: ["T1530"]
difficulty: "intermediate"
---

# AWS S3 Bucket Security Audit

## Overview
Comprehensive security assessment of S3 bucket configurations...

## Steps
1. List all S3 buckets
2. Check public access settings
3. Review bucket policies
```

Your AI agent reads the frontmatter for context, then executes the step-by-step workflows to perform security tasks like memory forensics, C2 beaconing detection, or Kubernetes RBAC audits.
